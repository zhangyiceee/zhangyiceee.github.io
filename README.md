1、1小时阅读文献+1000字的长摘要；

2、第二天，半个小时不看原文改1000字到200字的短摘要；再看原文，用半小时修改200字；

3、第三天，将200字改为100字（不看原文）

4、一个话题下至少阅读300篇相关文献
